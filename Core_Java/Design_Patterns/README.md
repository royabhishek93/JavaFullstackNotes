## Interview Prep Strategy

1. **Start here:** Q29 Singleton (most asked - 4 minutes)
2. **Then:** Q31 Builder (common in modern APIs - 4 minutes)
3. **Core patterns:** Q30, Q33 (Factory, Strategy - 6 minutes)

5. **Practice:** Implement one pattern from scratch (1-2 hours)
