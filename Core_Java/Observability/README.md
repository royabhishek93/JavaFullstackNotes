## Interview Prep Strategy

1. **Start here:** Q59 (logging frameworks - 3 minutes)
2. **Then:** Q63 (structured logging - 3 minutes)
3. **Metrics:** Q60 (Prometheus/Micrometer - 4 minutes)
4. **Advanced:** Q61, Q64 (tracing, aggregation - 8 minutes)
5. **Health:** Q62 (health checks - 3 minutes)
6. **Practice:** Add observability to a service (2-3 hours)

## Interview Frequency

- **42%** - Q59 Logging frameworks
- **40%** - Q60 Metrics, Q62 Health checks
- **38%** - Q61 Distributed tracing, Q63 Structured logging
- **35%** - Q64 Log aggregation

## Key Topics Covered

- SLF4J facade with Logback/Log4j2
- Micrometer metrics (counters, gauges, timers)
- OpenTelemetry tracing (spans, trace context)
- Kubernetes health/readiness probes
- JSON structured logs
- ELK stack (Elasticsearch, Logstash, Kibana)
